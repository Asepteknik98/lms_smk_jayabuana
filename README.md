# Learning Management System (LMS) - SMK Jaya Buana

Aplikasi LMS berbasis PHP Native murni, MySQL, dan Bootstrap 5 dengan arsitektur modular, aman, serta responsif.

---

## 📋 Persyaratan Sistem
1. XAMPP (Apache, PHP 8.x, MySQL/MariaDB)
2. Composer (untuk pengelolaan dependensi Dompdf & PhpSpreadsheet)

---

## 🚀 Langkah Instalasi di XAMPP

1. **Clone/Salin Proyek**
   Salin folder `LMS-SMK-JAYA-BUANA` ke direktori htdocs XAMPP:
   `C:\xampp\htdocs\LMS-SMK-JAYA-BUANA`

2. **Instalasi Dependensi PHP**
   Buka terminal/CMD di folder proyek, lalu jalankan:
   ```bash
   composer install
